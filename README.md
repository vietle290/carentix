# 🚖 Carentix

A full-stack ride-hailing and transportation platform that connects passengers with nearby drivers in real time. Users can book rides, track vehicle availability on an interactive map, make online payments, and communicate with drivers through integrated video calling and live location updates.

## Note

This project is developed for educational and portfolio purposes.

Some third-party services (such as payment gateways, email notifications, maps, and video calling) may require valid API keys and configuration before they can be used in production.

## ✨ Features

### Passenger

- User registration and authentication
- Search pickup and drop locations
- Interactive map with route visualization
- Real-time nearby vehicle discovery
- Ride booking and management
- Online payment integration
- Booking history
- Live driver tracking
- Video call with driver

### Driver / Partner

- Driver onboarding workflow
- Vehicle registration and verification
- Document upload and KYC verification
- Real-time location sharing
- Accept and manage ride requests
- Ride status updates
- Earnings tracking

### Admin

- User management
- Driver verification
- Vehicle approval system
- Booking monitoring
- Platform analytics

---

## 🚀 Key Highlights

- Full-stack application built entirely with Next.js
- Real-time ride matching using Socket.IO
- Interactive route calculation and map visualization
- Online payment processing with VNPay and PayOS
- Video calling support using ZEGO Cloud
- Driver KYC and onboarding workflow
- Responsive UI optimized for desktop and mobile devices
